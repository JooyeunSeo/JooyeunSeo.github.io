# Intro

오늘 여러분은 HTTP 요청과 REST API에 대해 배운 내용을 활용해서 공개 API에서 나온 데이터를 이용하는 웹사이트를 구축하는 프로젝트를 하게 됩니다.
예를 들면, 앞에서 우리는 날씨 API를 이용해서 비 알리미 앱을 만들었고 ISS 추적기도 만들어보고 비트코인 시세도 알아보았습니다, 모두 공개 API를 이용해서 말이죠.
오늘 여러분은 재미있다고 생각하는 API를 이용해서 그 API를 기반으로 한 서비스나 웹사이트를 구축해봅니다.
