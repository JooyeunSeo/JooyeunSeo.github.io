# Intro

Flask를 이용한 블로그 웹사이트 구축에서 배운 내용을 이용해서 나만의 전자상거래 웹사이트를 구축해볼 겁니다. 여러분의 웹사이트는 장바구니와 결제 기능이 있어야 합니다.

판매용 상품을 전시하고 사용자의 결제를 받을 수 있어야 합니다.

로그인/등록 인증 기능이 있어야 합니다.

아래 링크에서 웹사이트 예시를 살펴보세요.

<https://store.waitbutwhy.com/>



Stripe API도 사용하실지 고려하셔야 합니다.

<https://stripe.com/docs/payments/checkout>

- **Templates:** Jinja2 (HTML rendering)
- **Frontend:** JavaScript
- **Database:** SQLite
- **Email:** smtplib, email.mime
