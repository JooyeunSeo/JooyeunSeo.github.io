## sort / sorted

```python
nums = [3, 1, 2]
nums.sort()                   # sort  : 원본 리스트 변경
sorted_nums = sorted(nums)    # sorted: 새 리스트 반환
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>[1, 2, 3]</samp>

### reverse

```python
nums = [3, 1, 2]
nums.sort(reverse=False)                  # 오름차순(기본값)
desc_nums = sorted(nums, reverse=True)    # 내림차순
```
<i class="fa-solid fa-right-from-bracket"></i>       
<pre>
[1, 2, 3]        
[3, 2, 1]
</pre>

### key

```python
nums = ["3", "1", "2"]
nums.sort(key=int)          # 문자열 -> 숫자 변환 기준 정렬
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>["1", "2", "3"]</samp>

```python
words = ["apple", "orange", "pear", "banana"]
words.sort(key=len)         # 문자열 길이 기준 정렬
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>["pear", "apple", "orange", "banana"]</samp>

```python
words = ["apple", "orange", "pear", "banana"]
# 1차 기준: 길이
# 2차 기준: 사전 순서
words.sort(key=lambda x: (len(x), x))
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>["pear", "apple", "banana", "orange"]</samp>

```python
tuples = [(1, 3), (1, 2), (2, 1)]
# 1차 기준: 튜플의 1번째 값
# 2차 기준: 튜플의 2번째 값
tuples.sort(key=lambda x: (x[0], x[1]))
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>[(1, 2), (1, 3), (2, 1)]</samp>

```python
tuples = [(1, 2), (1, 4), (2, 5)]
# 1차 기준: 튜플 1번째 값 오름차순
# 2차 기준: 튜플 2번째 값 내림차순(int 타입에 대해서만 -x 가능)
tuples.sort(key=lambda x: (x[0], -x[1]))
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>[(1, 4), (1, 2), (2, 5)]</samp>

## Hash Table + sort

```python
order = {'gold': 0, 'silver': 1, 'bronze': 2}
medals = ['silver', 'gold', 'bronze']
medals.sort(key=lambda x: order[x])
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>['gold', 'silver', 'bronze']</samp>

## Counter + sort

```python
from collections import Counter

nums = [1, 1, 2, 3, 1, 2]
cnt = Counter(nums)           # Counter({1: 3, 2: 2, 3: 1}) 생성
# 1차 기준: 등장 빈도 내림차순
# 2차 기준: 값 오름차순
nums.sort(key=lambda x: (-cnt[x], x))
```
<i class="fa-solid fa-right-from-bracket"></i> <samp>[1, 1, 1, 2, 2, 3]</samp>
